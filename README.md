## ⚠️ 重要说明 (Compatibility)

### 关于 Clash 版本的特别提示

此 Clash 订阅使用了高级特性，**仅支持**以下内核版本：

* ✅ **Clash Premium**
* ✅ **Clash Meta (Mihomo)**

❌ **不支持** 普通开源版 Clash 核心。如果您在使用 OpenClash、Clash Verge Rev 等客户端，请确保内核已切换至 Meta/Premium 版本，否则可能会导致配置加载失败或规则无效。
